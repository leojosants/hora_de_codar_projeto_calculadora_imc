# Aplicação desenvolvida em React JS

## Calculadora para Índice de Massa Corporal - IMC

Os cálculos são realizados a partir da `altura` e do `peso`.

### Executando aplicação

  - `cd calculadora-imc`
  
  - `npm install`
  
  - `npm run dev`

### Recursos utilizados

  - HTML
  - CSS
  - React JS
  - Vite
  - VSCode
